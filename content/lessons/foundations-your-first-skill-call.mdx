# Your First Skill Call

MARS exposes useful robot behaviors as callable skills. In practice, that means you can ask the robot to perform a higher-level action without rebuilding every motor command yourself.

<Callout type="concept" title="Skill calls">
A skill call is a structured request to the robot: do a named action with the right parameters, then report what happened.
</Callout>

## Why this matters

Skills let you work at the right level of abstraction. You still need to understand the sensing-decision-action loop, but you do not need to start by writing low-level control code for every experiment.
